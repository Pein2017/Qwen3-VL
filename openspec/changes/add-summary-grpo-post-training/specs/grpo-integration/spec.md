# grpo-integration Specification (Delta)

## MODIFIED Requirements

### Requirement: grpo-integration is deprecated for summary GRPO
The `grpo-integration` spec SHALL be treated as deprecated for summary-mode GRPO post-training and SHALL be superseded by the `summary-grpo-post-training` specification.

#### Scenario: Spec deprecation guidance
- **GIVEN** summary-mode GRPO work is planned
- **WHEN** consulting OpenSpec
- **THEN** `summary-grpo-post-training` is the authoritative spec and `grpo-integration` is treated as deprecated for this scope
